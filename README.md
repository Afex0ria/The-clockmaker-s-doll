# The-clockmaker-s-doll
Идея:Игра раскрывает, что механический город, который ты чинешь, — это метафора твоего организма, а каждый уровень отражает восстановление органа.  Жанр:3D-пазл-адвенчура в Clockpunk, Steampunk,Fusionpunk 

# Инструкция по запуску

Запуск Игры: Требования к Среде
Для того чтобы запустить нашу игру из исходного кода, вам потребуется установленный игровой движок Unity.

Пожалуйста, используйте Unity Hub для установки и управления версиями. Мы рекомендуем использовать версию [Unity 6000.2.13f1].

1) Убедитесь, что у вас установлен Unity Hub.

2) С помощью Unity Hub установите указанную версию движка.

3) Клонируйте репозиторий.

4) Откройте клонированную папку проекта через Unity Hub, и вы сможете запустить игру в редакторе Unity.
